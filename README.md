# WebPic

A native macOS app for optimizing images for the web — convert, compress, resize, generate responsive sets, and produce copy-paste `<picture>` / React / Next.js / Vue snippets.

- **Platform:** macOS 14+
- **Tech:** SwiftUI (+ AppKit interop), Swift Package
- **Distribution:** downloadable DMG/ZIP with auto-update (planned)

Status: **in design** — implementation plan and spec live under [`docs/superpowers/specs/`](docs/superpowers/specs/).
